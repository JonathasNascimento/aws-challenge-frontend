## Setup Frontend

Para esse projeto foi usado o Node na versão v12.22.11

Após o clone do projeto:

- Instalar as dependências do package.json

  `$ npm install`

- Build do projeto

  `$ npx ng serve -o`
  ou
  `$ ng serve -o`
